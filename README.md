# AI Tools & Templates Hub

Welcome to the **AI Tools & Templates Hub** – a premium, beautifully curated directory for modern AI utilities, coding assistants, and pre-built design templates.

## Overview

This project is divided into two equally powerful sections that run side-by-side:
1. **AI Tools Hub (`index.html`)**: Focuses on advanced AI models, writing bots, coding assistants, and productivity software.
2. **AI Templates Hub (`templates.html`)**: Focuses on design-to-code platforms, UI component libraries, no-code website builders, and full-stack app generators.

## Key Features

* **Dual Environment Navigation**: Seamlessly jump between the Tools Hub and Templates Hub via the header buttons (complete with a "NEW" badge) or the sidebar navigation.
* **Complex Filtering Engine**: Filter items by specific categories, tags, or pricing (Free, Freemium, Paid).
* **Live Search & Sort**: Instantly search for tools/templates by name or description. Sort items Alphabetically, by Top Rated, or Newest.
* **Persistent State**: Favorites and 5-Star Ratings are saved directly to your browser's local storage. Tool saves and Template saves are independent of each other!
* **Responsive Visuals**: A sleek, dark-mode default interface featuring glassmorphic headers, radiant hover sweeps on cards, and full mobile optimization. Includes a light-theme toggle.
* **Authentication Ready**: Basic UI flows for Login and Registration are established, keeping unauthorized users out until they sign in.

## Technologies Used
- HTML5 & CSS3 (Vanilla frontend with advanced custom CSS properties)
- Vanilla JavaScript (ES6+ for data mapping, state management, and DOM manipulation)
- FontAwesome (Icons)
- Google Fonts (Syne & DM Sans)

## Project Structure
- `index.html` & `script.js` - The core AI Tools directory.
- `templates.html` & `templates.js` - The newly added AI Templates directory.
- `style.css` - Global stylesheet dictating the premium look of both pages.
- `login.html/js` & `register.html/js` - Authentication flow.

## Installation & Usage
Simply open `index.html` in your browser. No server setup is required for the frontend unless you decide to expand the backend logic!
